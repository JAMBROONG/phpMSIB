Halo! Nama saya Akhmad Lylana, Saya lahir di Cirebon 19 September 2000. Saya sangat menyukai programming karena sangat menyenanggan bisa menyusun kode menjadi sebuah program yang utuh. Saya mulai mengenal programming sejak saya menempuh pendidikan di SMK Yanuda Tama Munjul Cirebon. Kemudian Saya melanjutkan pendidikan Saya di Pesantren Teknologi Informasi dan Komunikasi (PeTIK) Depok untuk memperdalam pengetahuan saya di bidang programming. Satu tahun kemudian Saya lulus dan melanjutkan ke perguruan tinggi STT Terpadu Nurul Fikri Depok hingga saat ini saya sudah menjalani semester ke 5 di perguruan tinggi ini

Saat ini Saya sedang menjalankan program MSIB Kampus Merdeka batch ke 3. Saya sangat senang karena saya dapan menambah pengetahuan saya melalui program ini. Semoga Saya dan seluruh peserta MSIB Kampus Merdeka lainnya dapat menjalankan program ini dengan sebaik mungkin dan dapat bermanfaat bagi bangsa dan negara kelak. Dan untuk para mentor dan pembimbing kami semoga Allah memberikan kekuatan agar dapat terus membimbing kami dalam mengikuti program ini. Terimakasih untuk semua pihak terkait yang sudah menjalankan program ini.
